# Infra — Terraform

This directory provisions:

- A regional VPC with a single subnet, plus secondary ranges for Pods and Services.
- A regional GKE cluster in either **Autopilot** or **Standard** mode (controlled by `var.gke_mode`).
- A Google ServiceAccount + IAM bindings for the example workload, ready to be used via Workload Identity.

## Usage

```bash
just tf-init
just tf-plan
just tf-apply
```

## State backend

A GCS backend block is included in `versions.tf` but commented out. Uncomment and set the bucket once your state bucket exists. See `docs/development.md` for the bootstrap commands.
